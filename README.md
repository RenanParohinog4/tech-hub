# DevEvents

DevEvents is a modern event discovery platform built with Next.js that helps developers find, explore, and join technology events across the Philippines. Users can browse upcoming conferences, meetups, hackathons, workshops, and community gatherings, while organizers can create and manage events through an intuitive dashboard.

## Features

- 📅 Browse upcoming tech events
- 🔍 Search and discover events
- 🏷️ Filter events by category and tags
- 📝 Create and manage events
- 🖼️ Image upload with Cloudinary
- 📱 Fully responsive design
- ⚡ Built with the Next.js App Router

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- MongoDB
- Mongoose
- Cloudinary
